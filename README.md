## Overview

This project focuses on identifying and analyzing phishing emails using cybersecurity awareness techniques and phishing indicators.

---

## Objectives

- Detect phishing attempts
- Analyze suspicious email characteristics
- Identify phishing indicators
- Recommend prevention strategies

---

## Tools Used

- Browser
- Canva
- Screenshot Tools

---

## Key Findings

- Suspicious sender addresses
- Fake login links
- Urgent social engineering tactics
- Grammar and formatting issues

---

## Recommendations

- Verify sender addresses
- Avoid suspicious links
- Enable multi-factor authentication
- Never share sensitive information
